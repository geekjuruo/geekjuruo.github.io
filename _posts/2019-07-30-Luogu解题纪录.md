## P1009 阶乘之和

用高精度计算出S=1!+2!+3!+…+n!(n≤50)其中“!”表示阶乘，例如：5!=5×4×3×2×1。

输入格式：

一个正整数N

输出格式：

一个正整数S，表示计算结果。

示例:

```
无
```

### 思路

* 高精度问题。
* python打表也可。

### Solution

```java
#include<iostream>
#include<bits/stdc++.h>
using namespace std;
int a[2000];
int b[2000];
int c[2000];
int sum[2000];
void pplus(int *a,int *c)
{
    int jw=0;
    for(int i=1;i<=1000;i++)
    {
        c[i]+=a[i]+jw;
        jw=c[i]/10;
        c[i]%=10;
    }
}
void cheng(int *a,int c)
{
    int jw=0;
    for(int i=1;i<=1000;i++)
    {
        a[i]=a[i]*c+jw;
        jw=a[i]/10;
        a[i]%=10;
    }
}
int main()
{
    int n;
    cin>>n;
    a[1]=1;
    for(int i=1;i<=n;i++)
    {
        cheng(a,i);
        pplus(a,c);
    }
    bool flag=0;
    for(int i=1000;i>=1;i--)
    {
        if(c[i]!=0) flag=1;
        if(flag) cout<<c[i];
    }
}
```
